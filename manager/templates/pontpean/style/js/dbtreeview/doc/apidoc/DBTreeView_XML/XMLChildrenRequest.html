  • ChildrenRequest

XML Children request.

Request message send by the client. For example, when the user opens a branch, the following message could be :

  1.  <childrenRequest>
  2.    <attribute name="id" value="02">
  3.    <attribute name="filter" value="groupB">
  4.  </childrenRequest>
