Éléments de formatage de type bloc :

Paragraphe
Code :
<p>Texte dans le premier paragraphe.</p><p>Texte dans le second paragraphe.</p>
Résultat :

Texte dans le premier paragraphe.

Texte dans le second paragraphe.

Ligne
Code :
<hr />
Résultat :
Liste à puces
Code :
<ul>
<li>Premier élément de la liste</li>
<li>Second élément de la liste</li>
<li>...</li>
<ul>
Résultat :
  • Premier élément de la liste
  • Second élément de la liste
  • ...
Liste numérotée
Code :
<ol>
<li>Premier élément de la liste</li>
<li>Second élément de la liste</li>
<li>...</li>
<ol>
Résultat :
  1. Premier élément de la liste
  2. Second élément de la liste
  3. ...
Liste de définitions
Code :
<dl>
<dt>Premier mot à définir</dt>
<dd>Définition du premier mot</dd>
<dt>Second mot à définir</dt>
<dd>Définition du second mot</dd>
</dl>
Résultat :
Premier mot à définir
Définition du premier mot
Second mot à définir
Définition du second mot
Tableau
Code :
 
<table>
<caption>La légende du tableau</caption>
<tr>
<th>Premier titre de colonne</th>
<th>Second titre de colonne</th>
</tr>
<tr>
<td>Ligne 1, colonne 1</td>
<td>Ligne 1, colonne 2</td>
</tr>
<tr>
<td>Ligne 2, colonne 1</td>
<td>Ligne 2, colonne 2</td>
</tr>
</table>
Résultat :
La légende du tableau
Premier titre de colonne Second titre de colonne
Ligne 1, colonne 1 Ligne 1, colonne 2
Ligne 2, colonne 1 Ligne 2, colonne 2
Niveau de sous-titre x
Code :
<hx>Titre de niveau x</hx> (où x est compris entre 3 et 6)
Exemple :
<h5>Titre de niveau 5</h5>
Titre de niveau 5
Text préformaté
Code :
<pre>
Ce code sera affiché
comme il est avec les retours à la ligne
et la tabulation.
</pre>
Résultat :
Ce code sera affiché
comme il est avec les retours à la ligne
et la tabulation.
Bloc de citation
Code :
<blockquote>
<p>Ceci est un bloc de citation</p>
</blockquote>
Résultat :

Ceci est un bloc de citation

Éléments de formatage de type inline :

En gras
Code :
Ceci est un <strong>texte en gras</strong>
Résultat :
Ceci est un texte en gras
En italique
Code :
Ceci est un <em>texte en italique</em>
Résultat :
Ceci est un texte en italique
Retour à la ligne
Code :
Ceci est la première ligne <br />et ceci est la seconde ligne.
Résultat :
Ceci est la première ligne
et ceci est la seconde ligne.
Lien
Code :
Ceci est un lien pointant vers <a href="http://pxsystem.sourceforge.net/fr/"
title="Aller au site de Plume CMS" hreflang="fr">le site de Plume CMS</a>.
Résultat :
Ceci est un lien pointant vers le site de Plume CMS.
Note : L'attribut title contient le texte s'affichant dans l'infobulle lorsque l'on passe la souris au-dessus du lien.
Image
Code :
<img src="http://pxsystem.sourceforge.net/media/img/logo.png"
alt="Logo de plume CMS" style="width: 100px; height: 45px;" />
Résultat :

Note : L'attribut alt est le texte alternatif au cas où l'image ne peut être affichée.
Code
Code :
<code>var formatField = null;</code> : ceci est du JavaScript.
Résultat :
var formatField = null; : ceci est du JavaScript.
Citation
Code :
<q>Être ou ne pas être...</q>
Résultat :
Être ou ne pas être...
Acronym
Code :
<acronym title="HyperText Markup Language">HTML</acronym>
Résultat :
HTML
