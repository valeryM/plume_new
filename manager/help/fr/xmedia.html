Ajouter un fichier

Afin de pouvoir afficher une image ou créer un lien vers un document téléchargeable, à partir d'un article ou d'une brève, vous devez tout d'abord stocker sur le serveur de votre site le fichier correspondant. C'est justement le rôle de ce gestionnaire : vous permettre de transférer facilement les fichiers stockés sur votre ordinateur vers le site.

  • Choisissez tout d'abord le dossier dans lequel vous souhaitez ajouter le fichier. Si nécessaire, vous pouvez créer un sous-dossier.
  • Cliquez sur le bouton Parcourir... pour faire apparaître l'explorateur de fichier de votre ordinateur, allez à l'emplacement du fichier désiré, et selon votre système, sélectionnez-le et cliquez sur le bouton Ouvrir ou appuyez sur la touche Entrée, ce qui provoquera la fermeture de l'explorateur.
  • Ensuite il ne reste plus qu'à cliquer sur le bouton Ajouter un fichier pour le télécharger vers le serveur, dans le dossier que vous aurez sélectionné.

Le temps nécessaire au téléchargement dépend de votre type de connexion.
Faites attention à ne pas dépasser la taille de fichier autorisée et de bien utiliser des noms de fichiers sans espaces.

Créer un sous-dossier

Au fur et à mesure de l'évolution de votre site, vous serez amené à gérer des quantités de plus en plus importantes de fichiers divers : images, PDF, archives, applications... Si vous les regroupez tous dans le dossier par défaut, il deviendra rapidement laborieux d'y retrouver le fichier désiré. Pour éviter cela, vous pouvez créer, à partir du dossier par défaut, autant de sous-dossiers que vous le souhaitez, vous permettant alors de disposer d'un classement par thème ou catégorie.
