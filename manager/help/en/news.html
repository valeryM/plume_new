Category of the news

When you create a new news, you need to put it into a category. You can imagine that like putting a news into a folder to group them by themes.

A news can be part of multiple categories but it has only one Main category. When you create a news, the first category you select will be the Main category. After saving the news for the first time, you will have the possibility to put the news in more categories and also change the main category to another. These changes are done with the little icons in the list of categories of the news.

Format of the news

The format is the way you want to type in the content of the news. You can provide directly HTML content. This is complex, but it provides a lot of flexibility. You can also provide the content in an alternative syntax, the so called Wiki syntax. The Wiki syntax is a simplified but powerfull way to provide the content of your news. It is recommended to always use the Wiki syntax if you do not have special needs.

Status of the news

The status of a news defines for example if the news can be seen by the visitors on the website. Depending on your rights you can set the news as:

On-Line
The news can be seen on the website.
Off-Line
The news is kept in the database but not displayed.
In edition
The author (you) is working on the news.
Waiting for validation
The author (you) has finished working on the news and is waiting a reviewer to validate it.

Status of the comments

The status of the comments defines if the ressource should accept comments by the visitors on the website or not. Depending on your rights you can set this status to:

Comments open
The comment form is displayed on the resource page.
Comments closed
The comment form is not displayed, no comments can be added.

Title of the news

This should be self explanatory. The title of the news will be displayed on all the pages refering to the news.

Content of the news

The content of the news is provided in the format you have selected.

Keywords of the news

The keywords are a list of comma delimited words. For example you can put: Monotremata, Tachyglossidae, Kollikodontidae, Platanistoidae, river dolphins. The keywords can be composed of 2 words like river dolphins.

The keywords are of course used in the internal search engine, but also in the news page to give more details about the page for the global search engines like Google.

Associated website title

When creating a news, a reference website can be provided if available. Just put here the title.

Associated website address

Just put reference website address. The address must start with http://. Alternatively, you can put a link to file on a FTP file server with a link starting with ftp://.

Publication date

The publication date it the date and time at which the news start to be available on the website. If you give a date and time in the future, you can publish a news while you are away.

Expiration date

The expiration date it the date and time at which the news stop to be available on the website. You can use this date to create for example a news for a limited time for a special event. It is recommended not to abuse of it, as visitors through search engines like not to fall on pages that are not here anymore.
