Κατηγορία άρθρου

Όταν δημιουργείτε ένα άρθρο, πρέπει να το βάλετε σε μια κατηγορία. Για να γίνει πιο κατανοητό αυτό φανταστείτε πως βάζετε τα άρθρα σε ένα κατάλογο ανάλογα με το θέμα τους για να τα ταξινομήσετε.

Ένα άρθρο μπορεί να ανήκει σε διάφορες κατηγορίες αλλά μπορεί να έχει μόνο μια κύρια κατηγορία. Όταν δημιουργείτε ένα άρθρο, η πρώτη κατηγορία που θα επιλέξετε θα είναι και η κύρια κατηγορία. Αφού αποθηκεύσετε το άρθρο, έχετε τη δυνατότητα να το κατατάξετε σε περισσότερες κατηγορίες καθώς και να αλλάξετε την κύρια κατηγορία. Αυτές οι αλλαγές γίνονται από τα μικρά εικονίδια στη λίστα με τις κατηγορίες του άρθρου.

Τύπος του άρθρου

Ο τύπος έχει να κάνει με τον τρόπο που θέλετε να εισάγετε το περιεχόμενο του άρθρου. Μπορείτε να δώσετε άμεσα περιεχόμενο HTML το οποίο είναι κάπως περίπλοκο αλλά και πολύ ευέλικτο στην προσαρμογή του. Μπορείτε επίσης να παρουσιάσετε το περιεχόμενο με μια εναλλακτική σύνταξη, την περιώνυμη σύνταξη Wiki. Η σύνταξη Wiki είναι ένας απλοποιημένος αλλά πανίσχυρος τρόπος να αποδώσετε το περιεχόμενο του άρθρου σας. Προτείνουμε να τη χρησιμοποιείτε πάντα, εκτός και αν υπάρχουν ειδικοί λόγοι για να το αποφύγετε.

Η περιγραφή και κάθε ξεχωριστή σελίδα ενός άρθρου έχουν το δικό τους τύπο. Έτσι θα μπορείτε για παράδειγμα να χρησιμοποιήσετε τη σύνταξη Wiki για τα πάντα εκτός από εκείνη τη συγκεκριμένη σελίδα που πρέπει να βάλετε περίπλοκο κώδικα HTML.

Κατάσταση του άρθρου

Η κατάσταση ενός άρθρου καθορίζει αν και πως ένα άρθρο θα είναι διαθέσιμο στους επισκέπτες. Ανάλογα με τα δικαιώματά σας μπορείτε να ορίσετε την κατάσταση ενός άρθρου ως :

Σε σύνδεση
Το άρθρο θα είναι ορατό στον ιστότοπο.
Χωρίς σύνδεση
Το άρθρο καταχωρείται στη βάση δεδομένων αλλά δεν παρουσιάζεται στον ιστότοπο.
Σε επεξεργασία
Ο συντάκτης (δηλαδή εσείς) επεξεργάζεται ακόμα το άρθρο.
Σε αναμονή για επαλήθευση
Ο συντάκτης (δηλαδή εσείς) έχει τελειώσει την επεξεργασία και περιμένει επαλήθεθση από κάποιον που θα κάνει αξιολόγηση.

Σημείωση : Ακόμα κι αν είστε ο διαχειριστής δεν μπορείτε να ορίσετε την κατάσταση ενός άρθρου «σε σύνδεση» χωρίς να υπάρχει έστω μια σελίδα. Πρέπει να αποθηκεύσετε πρώτα το άρθρο αφού ορίσετε την κατάσταση του σε επεξεργασία, να προσθέσετε μια σελίδα και μετά να ορίσετε την κατάσταση του «σε σύνδεση».

Κατάσταση σχόλιων

Η κατάσταση των σχόλιων καθορίζει αν ο πόρος θα δέχεται σχόλια από τους επισκέπτες του ή όχι. Ανάλογα με τα δικαιώματα που έχετε μπορείτε να ορίσετε την κατάσταση αυτή ως:

Σχόλια ανοιχτά
Θα εμφανίζεται η φόρμα σχολιασμού στη σελίδα του πόρου.
Σχόλια κλειστά
Δεν θα εμφανίζεται η φόρμα σχολιασμού, δεν μπορούν να προστεθούν σχόλια.

Τίτλος άρθρου

Δεν χρειάζεται επεξήγηση. Ο τίτλος του άρθρου θα εμφανίζεται σε όλες τις σελίδες. Επιλέγοντας ένα ξεχωριστό τίτλο για κάθε άρθρο θα μπορείτε να ταξινομήσετε τα άρθρα μιας κατηγορίας.

Αριθμός σελίδας

Ας υποθέσουμε πως θέλετε να εμφανίζονται με μια συγκεκριμένη σειρά οι σελίδες του άρθρου σας στον ιστότοπο. Για παράδειγμα, έστω πως έχετε 3 άρθρα με τους τίτλους Πρωτάθλημα ποδοσφαίρου, Ομάδες και Βαθμολογία, στην κατηγορία Αθλητικά. Αν θέλετε να εμφανίζονται με αυτή τη σειρά στον ιστότοπο, απλά δώστε τους τίτλους ως : 1. Πρωτάθλημα ποδοσφαίρου, 2. Ομάδες και 3. Βαθμολογία.

Οι αριθμοί X. που υπάρχουν στην αρχή θα χρησιμοποιηθούν για την ταξινόμηση των άρθρων στην κατηγορία. Μπορείτε αργότερα, αν θέλετε, να αποκρύψετε τους αριθμούς από τις ρυθμίσεις της κατηγορίας.

Περιγραφή του άρθρου

Η περιγραφή του άρθρου χρησημοποιείται σαν στοιχείο στη λίστα της κατηγορίας και στην εμφάνισή του σε μηχανή αναζήτησης. Αυτή είναι και η περιγραφή που χρησιμοποιείται στο αρχείο ροής RSS. Η σύνταξη της περιγραφής γίνεται σύμφωνα με τον τύπο που έχετε επιλέξει. Μια καλή περιγραφή, συνήθως προκαλεί το ενδιαφέρον του αναγνώστη για να διαβάσει περισσότερα. Αν χρησιμοποιήσετε τη σύνταξη Wiki σαν τύπο μορφοποίησης της περιγραφής είναι ευκολότερο να δώσετε έμφαση στις λέξεις - κλειδιά.

Η περιγραφή χρησιμοποιείται επίσης και στις μηχανές αναζήτησης, μάλιστα οι λέξεις που υπάρχουν στην περιγραφή είναι πιο σημαντικές για την κατάταξη στη μηχανή αναζήτησης από τις λέξεις που υπάρχουν στο περιεχόμενο του άρθρου.

Λέξεις - κλειδιά του άρθρου

Οι λέξεις κλειδιά είναι μια λίστα λέξεων που διαχωρίζονται από κόμματα. Για παράδειγμα μπορείτε να δώσετε : ομάδα, αγώνας, βαθμολογία, κατάταξη, πρωτάθλημα ποδοσφαίρου. Οι λέξεις - κλειδιά μπορεί να είναι και οποιοσδήποτε συνδυασμός δύο λέξεων όπως πχ. πρωτάθλημα ποδοσφαίρου.

Οι λέξεις - κλειδιά χρησιμοποιούνται φυσικά στη μηχανή αναζήτησης του ιστότοπου αλλά δίνουν και περισσότερες πληροφορίες σε μεγάλες μηχανές αναζήτησης όπως το Google.

Διαδρομή του άρθρου

Η διαδρομή του άρθρου είναι δεδομένο καθοριστικής σημασίας αφού θα εμφανίζεται στη γραμμή διευθύνσεων του περιηγητή των επισκεπτών σας. Πρέπει να είναι σύντομη, ευκολομνημόνευτη, και καλογραμμένη. Για παράδειγμα, αν το άρθρο σας είναι στην κατηγορία Αθλητικά με τη διαδρομή της κατηγορίας να είναι /athlitika/ και δημιουργήσετε ένα άρθρο που η διαδρομή του θα είναι omades, οι επισκέπτες σας θα έχουν πρόσβαση σ΄αυτό στη διεύθυνση http://dieythynsh.com/athlitika/omades.

Στην περίπτωση που ένα άρθρο έχει πολλαπλές σελίδες, κάθε σελίδα θα έχει πρόσβαση με τη διαδρομή συν τον αριθμό της σελίδας. Για παράδειγμα, για να μεταβείτε στην τρίτη σελίδα του άρθρου μπορείτε να δώσετε τη διεύθυνση http://dieythynsh.com/athlitika/omadew3. Σημείωση : Η διαδρομή μπορεί να έχει μόνο λατινικούς χαρακτήρες.

Θα δείτε πως εξ ορισμού, καθώς τον πληκτρολογείτε τίτλο, ενημερώνεται και η διαδρομή. Μπορεί το αλφαριθμητικό να σας φανεί πολύ μεγάλο αλλά σας βοηθάει να επιλέξετε το καλύτερο. Κρατήστε το πολύ 3 λέξεις και διαχωρίστε τις με την κάτω παύλα (_) ή το σύμβολο μείον (-). Για παράδειγμα μπορείτε να δώσετε :

  • protathlima-podosfairou
  • bathmologia-superleague
  • athlitika-basket-a1

Να έχετε υπόψη σας πως αυτές οι λέξεις είναι οι πιο σημαντικές για τις μηχανές αναζήτησης.

Ημερομηνία δημοσίευσης

Η ημερομηνία δημοσίευσης είναι η ημερομηνία και η ώρα που η είδηση θα εμφανιστεί στον ιστότοπο. Αν δώσετε μια μελλοντική ημερομηνία, τότε η είδηση θα δημοσιευθεί αυτόματα τη δεδομένη χρονική στιγμή.

Ημερομηνία λήξης

Η ημερομηνία λήξης είναι η ημερομηνία και η ώρα που θα πάψει να εμφανίζεται η είδηση στον ιστότοπο. Μπορείτε να χρησιμοποιήσετε αυτή τη δυνατότητα για παράδειγμα όταν θέλετε να δημιουργήσετε μια είδηση που θα εμφανίζεται για περιορισμένο χρόνο σε μια ειδική περίσταση. Σας συμβουλεύουμε να μην κάνετε κατάχρηση αυτής της δυνατότητας γιατί δεν αρέσει και τόσο στους χρήστες που θα σας επισκεφθούν από μηχανές αναζήτησης να πέφτουν πάνω σε σελίδες που δεν υπάρχουν πια.

Αριθμός σελίδας

Όταν επεξεργάζεστε μια σελίδα ενός άρθρου, μπορείτε να αλλάξετε τη θέση της σε σχέση με τις άλλες σελίδες. Για παράδειγμα, αν θέλετε μπορείτε να κάνετε την τέταρτη σελίδα του άρθρου δεύτερη. Οι αριθμοί των υπόλοιπων σελίδων θα αλλαχθούν αυτόματα ώστε να προσαρμοστούν στην αλλαγή της σελίδας 4 στη θέση της σελίδας 2.

Όταν δημιουργείτε μια νέα σελίδα, η προεπιλογή είναι να μπαίνει στο τέλος του άρθρου (σαν τελευταία σελίδα).

Τίτλος σελίδας

Κάθε σελίδα ενός άρθρου, ακόμα κι αν το άρθρο έχει μόνο μια σελίδα, πρέπει να έχει τίτλο. Οι σελίδες ενός άρθρου μπορούν να θεωρηθούν σαν κεφάλαια του άρθρου, οπότε κάθε κεφάλαιο θα πρέπει να έχει τον τίτλο του.

Περιεχόμενο σελίδας

Το περιεχόμενο της σελίδας είναι και ο κύριος στόχος του άρθρου σας. Μπορείτε να επιλέξετε τη χρήση σύνταξης HTML ή Wiki για να το εισάγετε. Μην αμελείτε να το ελέγχετε διεξοδικά με τη χρήση της «οπτικοποίησης» πριν το αποθηκεύσετε. Η ποιότητα της γραφής σας θα κάνει τους επισκέπτες να επανέλθουν αν θέλουν να πάρουν πληροφορίες, αλλά και για να διαβάσουν ένα καλογραμμένο κείμενο.

Το κείμενό σας μπορεί να γίνει πιο ενδιαφέρον με την προσθήκη εικόνων και την παράθεση παραπομπών με δεσμών στο δίκτυο.
