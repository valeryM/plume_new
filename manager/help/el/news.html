Κατηγορία είδησης

Όταν δημιουργείτε μια είδηση, πρέπει να την κατατάξετε σε μια κατηγορία. Για να γίνει πιο κατανοητό αυτό φανταστείτε πως βάζετε τις ειδήσεις σε ένα κατάλογο ανάλογα με το θέμα τους για να τις ταξινομήσετε.

Μια είδηση μπορεί να ανήκει σε πολλές κατηγορίες αλλά μπορεί να έχει μόνο μια κύρια κατηγορία. Όταν δημιουργείτε μια είδηση, η πρώτη κατηγορία που θα επιλέξετε θα είναι η κύρια κατηγορία. Αφού αποθηκεύσετε την είδηση, έχετε τη δυνατότητα να την κατατάξετε σε περισσότερες κατηγορίες ή ακόμα και να αλλάξετε την κύρια κατηγορία. Αυτές οι αλλαγές μπορούν να γίνουν από τα μικρά εικονίδια στη λίστα των κατηγοριών των ειδήσεων.

Τύπος είδησης

Ο τύπος έχει να κάνει με τον τρόπο που θέλετε να εισάγετε την είδηση. Μπορείτε να της δώσετε άμεσα περιεχόμενο HTML το οποίο είναι κάπως περίπλοκο αλλά και πολύ ευέλικτο στην προσαρμογή του. Μπορείτε επίσης να παρουσιάσετε το περιεχόμενο με μια εναλλακτική σύνταξη, την περιώνυμη σύνταξη Wiki. Η σύνταξη Wiki είναι ένας απλοποιημένος αλλά πανίσχυρος τρόπος να αποδώσετε το περιεχόμενο της είδησής σας. Προτείνουμε να τη χρησιμοποιείτε πάντα, εκτός και αν υπάρχουν ειδικοί λόγοι για να το αποφύγετε.

Κατάσταση της είδησης

Η κατάσταση της είδησης ορίζει για παράδειγμα το αν ή είδηση θα είναι ορατή από τους επισκέπτες του ιστότοπου. Ανάλογα με τα δικαιώματα που έχετε, μπορείτε να ορίσετε την κατάσταση ως :

Σε σύνδεση
Η είδηση θα είναι ορατή στον ιστότοπο.
Χωρίς σύνδεση
Η είδηση καταχωρείται στη βάση δεδομένων αλλά δεν παρουσιάζεται στον ιστότοπο.
Σε επεξεργασία
Ο συντάκτης (δηλαδή εσείς) επεξεργάζεται ακόμα την είδηση.
Σε αναμονή για επαλήθευση
Ο συντάκτης (δηλαδή εσείς) έχει τελειώσει την επεξεργασία της είδησης και περιμένει να την επαληθεύσει κάποιος που θα κάνει αξιολόγηση.

Κατάσταση σχόλιων

Η κατάσταση των σχόλιων καθορίζει αν ο πόρος θα δέχεται σχόλια από τους επισκέπτες του ή όχι. Ανάλογα με τα δικαιώματα που έχετε μπορείτε να ορίσετε την κατάσταση αυτή ως:

Σχόλια ανοιχτά
Θα εμφανίζεται η φόρμα σχολιασμού στη σελίδα του πόρου.
Σχόλια κλειστά
Δεν θα εμφανίζεται η φόρμα σχολιασμού, δεν μπορούν να προστεθούν σχόλια.

Τίτλος είδησης

Δεν χρειάζεται επεξήγηση. Ο τίτλος της είδησης θα εμφανίζεται σε όλες τις σελίδες που θα υπάρχει η είδηση.

Περιεχόμενο της είδησης

Το περιεχόμενο της είδησης παρέχεται με τη μορφοποίηση που έχετε επιλέξει.

Λέξεις - κλειδιά της είδησης

Οι λέξεις - κλειδιά είναι μια λίστα λέξεων που διαχωρίζονται από κόμματα. Για παράδειγμα μπορείτε να δώσετε : ομάδα, αγώνας, βαθμολογία, κατάταξη, πρωτάθλημα ποδοσφαίρου. Οι λέξεις - κλειδιά μπορεί να είναι και οποιοσδήποτε συνδυασμός δύο λέξεων όπως πχ πρωτάθλημα ποδοσφαίρου.

Οι λέξεις - κλειδιά χρησιμοποιούνται φυσικά στη μηχανή αναζήτησης του ιστότοπου αλλά δίνουν και περισσότερες πληροφορίες σε μεγάλες μηχανές αναζήτησης όπως το Google.

Τίτλος σχετικού ιστότοπου

Όταν δημιουργείτε μια είδηση, μπορείτε να παραθέσετε κι ένα ιστότοπο αναφοράς της, αν υπάρχει. Απλά δώστε εδώ τον τίτλο του.

Διεύθυνση σχετικού ιστότοπου

Απλά δώστε τη διεύθυνση του σχετικού ιστότοπου. Η διεύθυνση θα πρέπει να ξεκινάει από http://. Εναλλακτικά, μπορείτε να βάλετε δεσμό προς ένα αρχείο σε διακομιστή FTP και στην περίπτωση αυτή θα πρέπει να ξεκινάει από ftp://.

Ημερομηνία δημοσίευσης

Η ημερομηνία δημοσίευσης είναι η ημερομηνία και η ώρα που η είδηση θα εμφανιστεί στον ιστότοπο. Αν δώσετε μια μελλοντική ημερομηνία, τότε η είδηση θα δημοσιευθεί αυτόματα τη δεδομένη χρονική στιγμή.

Ημερομηνία λήξης

Η ημερομηνία λήξης είναι η ημερομηνία και η ώρα που θα πάψει να εμφανίζεται η είδηση στον ιστότοπο. Μπορείτε να χρησιμοποιήσετε αυτή τη δυνατότητα για παράδειγμα όταν θέλετε να δημιουργήσετε μια είδηση που θα εμφανίζεται για περιορισμένο χρόνο σε μια ειδική περίσταση. Σας συμβουλεύουμε να μην κάνετε κατάχρηση αυτής της δυνατότητας γιατί δεν αρέσει και τόσο στους χρήστες που θα σας επισκεφθούν από μηχανές αναζήτησης να πέφτουν πάνω σε σελίδες που δεν υπάρχουν πια.
