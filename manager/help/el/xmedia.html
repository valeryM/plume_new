Προσθέτoντας αρχεία

Για να μπορείτε να δημιουργήσετε δεσμούς σε αρχεία μέσα στα άρθρα ή στις ειδήσεις σας, πρέπει να υπάρχει το αρχείο αυτό στο διακομιστή. Για να το πετύχετε αυτό σας παρέχεται μια διασύνδεση για τη μεταφορά αρχείων από τον υπολογιστή σας προς το διακομιστή.

  • Επιλέξτε τον κατάλογο στον οποίο θέλετε να προστεθεί το αρχείο, μπορείτε αν θέλετε να δημιουργήσετε έναν υποκατάλογο σύμφωνα με τις επιθυμίες σας.
  • Κάντε κλικ στο κουμπί Αναζήτηση.... Θα εμφανιστεί ο διαχειριστής αρχείων που θα σας δώσει πρόσβαση στα αρχεία του υπολογιστή σας
  • Εντοπίστε το αρχείο που θέλετε. Επιλέξτε το και επιβεβαιώστε την επιλογή σας πατώντας το κουμπί Άνοιγμα ή απλά το Enter (ανάλογα το σύστημά σας). Έτσι θα κλείσει το παράθυρο του διαχειριστή αρχείων.
  • Τώρα κάντε κλικ στο κουμπί Προσθήκη αρχείου, αυτό θα ξεκκινήσει τη μεταφορά του αρχείου στον τρέχοντα κατάλογο του διακομιστή.

Ανάλογα με την πρόσβαση που έχετε στο διαδίκτυο, η μεταφορά μπορεί να διαρκέσει μερική ώρα, οπότε μπορεί να χρειαστεί να περιμένετε λίγο. Προσέξτε μόνο το όνομα του αρχείου σας να μην περιλαμβάνει κενά διαστήματα.

Δημιουργία υποκαταλόγου

Καθώς ο ιστότοπός σας θα μεγαλώνει, θα φτάσετε στο σημείο να πρέπει να διαχειριστείτε πολλά αρχεία, εικόνες, αρχεία pdf ή ακόμα και αρχεία εφαρμογών. Αν τα αποθηκεύετε όλα στο βασικό κατάλογο αρχείων και εικόνων, σύντομα θα φτάσετε στο σημείο που θα σας είναι πολύ δύσκολο να τα εντοπίσετε ξανά. Για να το αποφύγετε αυτό θα πρέπει να δημιουργήσετε υποκατάλογους μέσα στον βασικό κατάλογο εγγράφων ώστε να ταξινομήσετε τα αρχεία σας ανάλογα με την κατηγορία τους. Μπορείτε να δημιουργήσετε όσους υποκαταλόγους θέλετε.
